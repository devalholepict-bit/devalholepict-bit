### 🤖 AskMee.Ai — AI-Powered Web Search Chat App
[![Repo](https://img.shields.io/badge/GitHub-Repo-181717?style=for-the-badge&logo=github)](https://github.com/devalholepict-bit/AskMee.Ai)

</div>

> Real-time AI chat with **Socket.IO streaming**, **LangChain agent** (Mistral AI), and conditional **Tavily web search** — reducing redundant API calls by ~60%.

**Highlights:**
- 🔐 Zero-trust JWT HTTP-only cookie auth + Gmail OAuth2 email verification
- 👁️ Multimodal vision AI (Pixtral-12B) for image-to-text understanding
- 🌗 Global dark/light theme with CSS variables + React Context
- ⚡ Real-time bidirectional communication via Socket.IO

`React` `Node.js` `LangChain` `Mistral AI` `Socket.IO` `MongoDB` `JWT` `Nodemailer`

### 🎬 The Movie Point — Full-Stack Movie Discovery Platform
[![Repo](https://img.shields.io/badge/GitHub-Repo-181717?style=for-the-badge&logo=github)](https://github.com/devalholepict-bit/The-Movie-Point)
[![Live](https://img.shields.io/badge/Live-Demo-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://the-movies-point.vercel.app/)

</div>

> Full-stack movie platform integrating **TMDB REST API** with a **Redux Toolkit** global state architecture, role-based admin dashboard, and distributed Vercel/Render deployment.

**Highlights:**
- 🎭 TMDB API — real-time trending, search & trailer data
- 🛡️ Role-Based Access Control (RBAC) with bcryptjs + full CRUD admin panel
- ❤️ Personalized favorites & watch history with MongoDB persistence
- ⚡ Sub-200ms Redux state sync across all slices

`React` `Redux Toolkit` `Node.js` `Express` `MongoDB` `JWT` `TMDB API` `Vite`
